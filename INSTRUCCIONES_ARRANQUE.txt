PASOS PARA INICIAR EL SISTEMA HABANA GYM

1. Abrir una terminal (PowerShell o CMD).

2. Navegar a la carpeta del proyecto:
   cd c:\Users\haban\.gemini\antigravity\playground\perihelion-galaxy

3. Ejecutar el comando de inicio:
   npm run dev

4. Una vez que diga "calentando" o "ready", abrir el navegador en:
   http://localhost:3002

NOTAS:
- Si el puerto 3002 está ocupado, puede fallar. Asegúrate de no tener otra terminal corriendo lo mismo.
- Para detener el sistema, presiona Ctrl + C en la terminal.
